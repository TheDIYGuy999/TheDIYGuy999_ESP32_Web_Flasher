Welcome to the TheDIYGuy999 ESP32 Web Flasher!

This installer is suitable for most ESP32 boards with 4MB flash memory

  1. Connect your ESP32 device to a USB port
  2. Select the software you want to install
  3. Click "Connect" and then select the correct COM port. No ESP32 board found?
  4. The programming will take less than 3 minutes
